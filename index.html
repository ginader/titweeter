Titweeter by davglass

Titweeter is an Open Source Twitter client for Android based smart phones.

Screenshots

Install

Installing Titweeter is easy on an Android based phone.

Simply click this link: Install Titweeter

Note: You must set your phone to "Allow install of non-Market applications."

Settings > Applications > Unknown Sources
